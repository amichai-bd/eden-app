# מבחן ברכות ותפילה

אפליקציה סטטית בעברית לתרגול מבחן של 125 שאלות.

## מה יש באפליקציה

- מסך פתיחה בעברית.
- מבחן מלא של 125 שאלות.
- שמירת התקדמות בדפדפן.
- סיכום תוצאה בסוף.
- חזרה ממוקדת על כל השאלות שנענו לא נכון.

## הרצה מקומית

```bash
python -m http.server 8080
```

ואז לפתוח:

```text
http://localhost:8080
```
